# Analysis_of_customer_behavior_Project


An industry-standard, end-to-end data analytics project simulating a corporate-grade business intelligence workflow. This project covers data cleaning, relational database engineering, advanced SQL querying, and interactive dashboard design to convert raw e-commerce data into strategic business insights.

---

## 🛠️ Tech Stack & Skills
* **Data Cleaning & ETL:** Python (Pandas, NumPy)
* **Database & Engine:** PostgreSQL / SQLAlchemy / Psycopg2
* **Data Analysis:** SQL (Joins, CTEs, Aggregations, Window Functions)
* **Business Intelligence:** Power BI (DAX, Data Modeling, Dashboard Design)

---

## 📌 Project Architecture & Workflow

1. **Data Preparation & ETL (Python):** * Formatted, cleaned, and structurally prepared the raw customer data using Jupyter Notebooks.
   * Established an active pipeline connection from Python to a local PostgreSQL server using `SQLAlchemy` and `psycopg2`.
   * Automatically loaded the processed Pandas DataFrame directly into a relational SQL database.

2. **Advanced Database Querying (SQL):**
   * Designed a relational structure to store transaction records.
   * Wrote high-performance SQL queries to solve complex business questions regarding customer segmentation, purchasing drivers, and loyalty programs.

3. **Data Visualization & Dashboard (Power BI):**
   * Connected the live PostgreSQL database directly to Power BI.
   * Built interactive, stakeholder-ready dashboards monitoring critical sales metrics, performance KPIs, and customer behavioral trends.
